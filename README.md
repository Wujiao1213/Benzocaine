# 核心功能
支持完整的苯佐卡因制备实验；</br>
支持3D虚拟数字人导师，后台接入Coze智能体；</br>
支持教学模式、考试模式；</br>
支持VR/AR模式：Pico 4手柄操作（如倾倒液体、调节旋钮），适合沉浸式训练。</br>

# 所需插件
MetaHumanSDK：用于驱动虚拟数字人导师相关功能，支持 MetaHuman 角色集成、语音/表情/口型等交互表现能力。</br>
AsyncLoadingScreen：用于项目关卡与场景切换时显示异步加载界面，提升大型资源加载过程中的用户体验。</br>
