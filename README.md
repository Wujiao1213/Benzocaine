# 核心功能
支持完整的苯佐卡因制备实验；
支持3D虚拟数字人导师，后台接入Coze智能体；
支持教学模式、考试模式；
支持VR/AR模式：Pico 4手柄操作（如倾倒液体、调节旋钮），适合沉浸式训练。
